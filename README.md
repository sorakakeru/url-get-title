# url-get-title

URLからtitle要素の文言を取得する

## 取得書式

### タイトル文言＋URL（タイトル文言とURLの間に改行あり）

```txt
title
https://example.com
```

### リンク（Markdown形式）

```txt
[title](https://example.com)
```

### リンク（HTML形式）

```txt
<a href="https://example.com" target="_blank">title</a>
```

### タイトル文言のみ

```txt
title
```
